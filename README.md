# Neuropatia Marketing

Landing page informativa sobre neuropatía diabética, construida con Next.js
(App Router) y Tailwind CSS. Pensada para pacientes y sus familias, con enfoque
en educación, confianza médica y conversión hacia WhatsApp.

## Requisitos
- Node.js 18 o superior

## Uso
```bash
npm install
npm run dev
```

Luego abre `http://localhost:3000`.

## Personalizacion rapida
- Actualiza el número de WhatsApp y el mensaje en `components/WhatsAppButton.jsx`.
- Ajusta textos y precios en los componentes de `components`.
